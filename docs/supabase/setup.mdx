### Supabase Setup

1. Sign up for a [Supabase](https://supabase.com/) account
2. Create a new project
3. In your project dashboard, go to `Settings` > `API`
4. Copy your `Project URL` and `anon` public API key

You'll need to add these to your environment variables: (.env file)

```
NEXT_PUBLIC_SUPABASE_URL=your_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_public_api_key
```
